DESCRIPTION

This file defines the various procedures for drawing primitives (lines, rectangles, circles), textures, blitting rectangles, and texture caching/loading. Note that, as of 7/25/2020, there IS a distinction between blitting and drawing! Blitting refers to COPYING and altering a texture, whereas drawing simply means drawning with no crops or cuts. Alterations may exist, but there's no splicing like a sprite sheet would have.
