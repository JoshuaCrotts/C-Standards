{\footnotesize\ttfamily void Stds\+\_\+\+Particle\+System\+Update (\begin{DoxyParamCaption}\item[{struct \mbox{\hyperlink{structparticle__system__t}{particle\+\_\+system\+\_\+t}} $\ast$}]{ps }\end{DoxyParamCaption})}

Updates the particle system. The programmer is in charge of determining when a particle is dead or not. The system only updates and swaps the particles. When a particle p dies, it is swapped with the last particle in the system, so it won\textquotesingle{}t be updated any further. In summary, dead particles are at the right-\/side of the system, and alive particles are on the left.
