Updates the animation type. If it is a sprite sheet, it advances the coordinates used to keep track of the current sprite in the sheet. Oppositely, if it is a series of images, we just advance the pointer keeping track of each image. Once the cycle ends, the pointer or coordinate is reset back to 0.
